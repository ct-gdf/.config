# Dotfiles

This is my main `.config` directory that contains my neovim configuration to set up my Personal Development Environment (PDE)
for my day to day development work.

## Installation
1. [Install neovim](https://neovim.io/)

2. Clone this repo to your `~/.config`
3. Launch neovim 
```shell
nvim .
```
4. Within neovim install all the plugins with Lazy by entering the command:

```shell
:Lazy
```
Then press `I` to begin installlation
5. Restart neovim
```shell
:q
nvim .
```
6. Enjoy 😄
